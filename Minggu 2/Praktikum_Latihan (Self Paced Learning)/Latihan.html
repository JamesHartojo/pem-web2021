B. Paragraph

Ini adalah contoh paragraf dalam HTML yang dimulai dengan tag p dan diakhiri tag /p.


C. Comments

Comments tidak dapat terlihat di tampilan website dan digunakan untuk menyembunyikan suatu konten di website.
Comment dapat dilihat pada format element suatu website dengan klik F12.


D. Text Formatting

Terdapat 11 jenis format text di HTML :

  1. Tag b untuk text tebal : Bold text
  2. Tag strong untuk text penting : Important text
  3. Tag i untuk text miring : Italic text
  4. Tag em untuk text yang ditekankan : Emphasized text
  5. Tag mark untuk menandai text : Marked text
  6. Tag small untuk mengecilkan text : Smaller text
  7. Tag del untuk menunjukkan text yang dihapus/dicoret : Deleted text
  8. Tag ins untuk menggaris bawahi text : Inserted text
  9. Tag sub untuk menuliskan text dengan ukuran lebih kecil dan posisi di bawah : Subscript text
  10. Tag sup untuk menuliskan text dengan ukuran lebih kecil dan posisi di atas :Superscript text

L. Quotation and Citation

Ada beberapa atribut berdasarkan kegunaannya :

  • Tag blockquote, contoh :
    Blockquote adalah elemen untuk menunjukkan suatu konten dikutip dari suatu sumber.

  • Tag q untuk memberi tanda kutip pada suatu konten, contoh :
    Misi perusahaan tersebut adalah: Meningkatkan kesadaran masyarakat Indonesia akan sampah.

  • Tag abbr untuk menyisipkan kepanjangan dari suatu singkatan, contoh :
    Organisasi PBB, WHO

  • Tag address untuk menunjukkan informasi kontak pada website, contoh :
    Ditulis oleh James.H
    Kunjungi kami di :
    www.BestSite.com
    Semarang
    Indonesia

  • Tag cite untuk menandai suatu karya, contoh :
    Lagu Indonesia Raya merupakan lagu kebangsaan negara Indonesia.

  • Tag bdo untuk menyusun huruf dan kalimat ke arah sebaliknya, contoh :
    Kebalikan dari Universitas Dian Nuswantoro adalah Universitas Dian Nuswantoro.
